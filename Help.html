Help - Frequently Asked Questions

What is the Minecraft Server?

The Minecraft Server is a community-driven server where players can explore, build, and play together in a vast sandbox environment.

How can I join the server?

You can join the server by entering the server IP address in the multiplayer section of your Minecraft client. Check our website for the current IP.

What are the server rules?

Our server rules promote a friendly environment; no griefing, cheating, or harassment is allowed. Please read the full rules on our website.

How do I report an issue?

If you encounter any problems, please contact our support team through the support link found on our website.
